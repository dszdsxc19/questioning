## 三、JavaScript 代码题

### 异常处理与事件循环

9. **看代码说结果（setTimeout 里的 `throw 1` 在 catch 里能拿到吗？可以结合事件循环机制讲一下）**

```javascript
try {
  setTimeout(() => {
    throw 1
  }, 0)
} catch (ex) {
  console.log(ex)
}
```

<details>
<summary>答案</summary>

**结果：** catch 无法捕获到异常，程序会报错 `Uncaught Error: 1`

**原因：**
- `setTimeout` 的回调函数在事件循环的宏任务队列中执行
- `try-catch` 块在当前同步代码执行时就已结束
- 当宏任务执行抛出异常时，`catch` 块已经不在调用栈中了
</details>
