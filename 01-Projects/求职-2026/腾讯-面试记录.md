### 5. React 合成事件

**问题**：请说明 React 的合成事件的作用，与原生事件的区别是什么

**我的回答**：{{待补充}}

**补充要点**：

**作用**：
- 屏蔽底层浏览器差异，提供统一的事件 API
- 便于事件的统一管理和批量更新

**与原生事件的区别**：
| 特性 | 原生事件 | React 合成事件 |
|------|----------|----------------|
| 命名方式 | 全小写（onclick） | 驼峰（onClick） |
| 事件绑定 | 字符串 | 函数 |
| 阻止默认行为 | `return false` | `e.preventDefault()` |
| 事件流 | 捕获→目标→冒泡 | 统一在冒泡阶段触发（可配置） |
| 事件对象 | 原生 Event | SyntheticEvent（包装对象） |
| 事件委托 | 手动实现 | 自动批量处理 |

**设计思路**：

**核心功能**：
1. 路由配置：支持动态注册、修改、删除路由
2. 权限控制：基于角色/权限的路由访问控制
3. 菜单生成：根据路由动态生成导航菜单
4. 路由守卫：前置/后置钩子进行权限校验

**权限滥用解决方案**：
- **最小权限原则**：按需分配，避免过度授权
- **权限分层**：页面级 → 组件级 → 按钮级
- **后端校验**：所有接口调用必须验证权限，不依赖前端
- **审计日志**：记录权限变更和敏感操作
- **RBAC 模型**：用户 → 角色 → 权限，通过角色间接管理权限
- **权限缓存**：JWT + 权限标识，定期刷新

### 7. DOM 和 BOM 的区别和联系

**问题**：DOM BOM 区别和联系

**我的回答**：{{待补充}}

**补充要点**：

**DOM（文档对象模型）**：
- 作用：操作 HTML/XML 文档的结构、样式和内容
- 核心：document 对象
- 标准：W3C 规范
- 常用 API：`querySelector`、`createElement`、`addEventListener` 等

**BOM（浏览器对象模型）**：
- 作用：与浏览器环境交互
- 核心：window 对象
- 非标准：各浏览器实现有差异
- 常用 API：`location`、`history`、`localStorage`、`setTimeout` 等

**联系**：
- BOM 包含 DOM（window.document）
- 都是 JavaScript 访问浏览器功能的接口
- 共享全局执行环境

### 9. Webpack HMR 原理

**问题**：webpack HMR 是什么，原理是什么？

**我的回答**：{{待补充}}

**补充要点**：

**HMR（Hot Module Replacement）**：模块热替换，在应用运行时替换、添加、删除模块，无需刷新页面。

**原理**：
1. **Webpack Watch**：监听文件变化，重新编译
2. **WebSocket 通信**：webpack-dev-server 与浏览器建立 WS 连接
3. **Manifest 更新**：服务器发送更新的模块 hash
4. **JSONP 请求**：浏览器请求更新的模块代码
5. **HMR Runtime**：替换模块并执行 `module.hot.accept` 回调
6. **冒泡机制**：如果上层没有 accept 回调，继续向上冒泡

```javascript
// HMR 使用示例
if (module.hot) {
  module.hot.accept('./module', () => {
    console.log('模块已更新')
  })
}
```

### 10. React SSR 原理

**问题**：SSR React 服务端渲染的原理

**我的回答**：{{待补充}}

**补充要点**：

**核心思想**：在服务器将 React 组件渲染成 HTML 字符串，直接返回给浏览器。

**SSR 流程**：
1. 服务器接收请求
2. 获取数据（API 调用）
3. 渲染 React 组件为 HTML（`renderToString`）
4. 返回完整 HTML + 初始状态
5. 浏览器渲染 HTML
6. **Hydration（注水）**：React 接管页面，恢复事件和状态

**同构渲染**：
- 代码复用：客户端和服务端共用组件代码
- **Next.js/Nuxt.js**：成熟的 SSR 框架

**优缺点**：
- ✅ 首屏快、SEO 友好
- ❌ 服务器负载高、开发复杂度高

```javascript
// SSR 基本实现
import { renderToString } from 'react-dom/server'

app.get('/', (req, res) => {
  const html = renderToString(<App />)
  res.send(`
    <!DOCTYPE html>
    <html>
      <body>
        <div id="root">${html}</div>
        <script src="client.js"></script>
      </body>
    </html>
  `)
})
```
