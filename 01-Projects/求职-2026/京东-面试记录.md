## 算法题

### 题目 1：数组转树形结构
**题目**：数组转列表：`[{id:1, parent_id: 0}, {id:2, parent_id: 0}, {id:3, parent_id: 1}]`

**要求**：时间复杂度 O(n)

**我的回答**：没写出来 ❌

**参考解法**：
```javascript
// 使用 Map 建立 id -> node 的映射，O(n) 时间复杂度
function arrayToTree(arr) {
  const map = new Map()
  const result = []

  // 第一遍：建立映射
  for (const item of arr) {
    map.set(item.id, { ...item, children: [] })
  }

  // 第二遍：构建树
  for (const item of arr) {
    const node = map.get(item.id)
    if (item.parent_id === 0) {
      result.push(node)
    } else {
      const parent = map.get(item.parent_id)
      parent?.children.push(node)
    }
  }

  return result
}
```
