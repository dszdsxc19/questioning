# Lando Norris 官网案例观察

## Summary

这是一篇高完成度的个人品牌网站案例观察，价值主要在于品牌叙事、3D 交互和非常规布局的做法，但它目前更像案例素材，不是稳定主题页。

## 为什么先留在 Capture

- 这页主要记录单一案例，不适合作为 `03-Resources` 的 canonical page
- 它的长期价值来自可提炼的方法，而不是这个站点本身
- 目前 vault 里还没有“网页设计案例观察”或“个人品牌网站设计”这类承接页

## 可提炼的稳定点

- 个人品牌网站可以用行业配色延展成数字产品的视觉语言
- 3D 模型与 2D 界面的融合，适合强化记忆点，但前提是性能和叙事能兜住
- 不对称布局和全屏区块适合讲故事，不适合高密度信息传达
- 彩蛋、签名动效、年份时间线这类元素，本质是在做品牌人格化
