# 汇率倒差价 - 多币种套利思路

## 核心想法

倒差价的一种方式：倒汇率。

各国的货币汇率一直在波动，比如最近人民币升值，日币贬值（注意，都是以美元作为基准），美元也在升值。那么如果能用日币结算的一些服务，就会相当于便宜很多。

## 当前环境（2026年3月）

- 人民币偏强
- 美元强
- 日元极弱
- 全球通胀不低

## 几种玩法

### 1. 外币存款

**例子：**
2026 年 3 月：7.0 换汇 1 万美元，年利率 3%

若一年后汇率 6.9：本息 10300 美元→71070 元 → 收益≈1.5%（汇率 + 利息）
若汇率 6.7：本息→69010 元 → 收益≈-1.4%（利息被汇率吃掉）

### 2. QDII 基金

**QDII = 允许你用人民币，间接买海外资产的基金。**

展开说：
- 你在中国，用人民币买
- 基金公司帮你换成美元/港币/欧元
- 去买**美股、港股、日股、美债、海外房地产、黄金原油**等等
- 赚了以后再换回人民币给你

所以它的作用就是：
**不用出国、不用换汇、不用开境外账户，就能投资全世界。**

#### QDII 基金赚什么钱？

主要赚两部分：
1. **投资标的涨了**（比如美股涨、日股涨）
2. **汇率赚了**（比如美元升值 → 换回人民币更多）

反过来也会亏：
- 股市跌
- 人民币大涨，把收益吃掉

#### 普通人怎么"玩"QDII？

**在哪买？**
支付宝、天天基金、微信理财通、券商APP，全都能买，**10块钱起**。

**买哪种？（按风险从低到高）**

1. **稳健型：美元债 QDII**
   - 投美国国债、高等级债券
   - 波动小，年化大概 3%–5%
   - 适合想**拿美元利息、不想炒股**的人

2. **指数型：美股指数 QDII（最常用）**
   - 标普500、纳斯达克100
   - 长期收益稳，波动比A股小
   - 适合定投

3. **港股 QDII**
   - 投恒生科技、恒生指数
   - 波动大，弹性强
   - 适合风险承受高的

4. **日元/日本股市 QDII**
   - 现在日元极弱，日股又在高位
   - 适合博弈**日元反弹**

5. **商品类：黄金、原油 QDII**
   - 对冲通胀、对冲汇率
   - 不适合长期拿，适合波段

**最简单玩法：定投**
每月固定日期买一点，比如：
- 每月 500 元 纳指100 QDII
- 每月 300 元 美元债 QDII

长期下来，既能分散风险，又能赚汇率+股市双重收益。

#### 关键特点（必须懂，不然会亏）

1. **有汇率风险**
   - 美股涨了 5% + 人民币兑美元大涨 4% = 最后可能只赚 1%
   - 反过来：美股小涨 + 美元升值 = **收益加倍**

2. **到账超级慢**
   - A股基金：**T+1 到账**
   - QDII：**T+2 确认，卖出要 7～10 天才能到账**
   - 急用钱别买

3. **有额度限制**
   - 有时候外汇额度用完，基金就会**暂停买入 / 限额买入**

4. **时差问题**
   - 你白天买，按**当晚美股收盘价**算，不是实时

#### 现在适合怎么配？（结合当前汇率）

结合当前情况：**人民币偏强、美元强、日元极弱**

简单组合：
- 40% 美股指数 QDII（纳指/标普）
- 30% 美元债 QDII（稳利息）
- 20% 港股 QDII（弹性）
- 10% 黄金 QDII（对冲）

如果你想博弈**日元反弹**：可以加一点 **日本股市 QDII**，但不要超过10%。

#### 哪些坑千万别踩？

1. **不要短期频繁买卖**（手续费高、到账慢，短线必亏）
2. **不要全仓 QDII**（最多占你总资产 20%–30%）
3. **不要碰高风险杠杆 QDII**（有些是油气杠杆、原油基金，一天能亏20%）
4. **不要在人民币大涨时重仓换美元资产**（收益会被汇率吃掉）

### 3. 海外订阅 / 消费套利

**当前最优：日元结算**

逻辑：人民币强 + 美元强 + 日元极弱 → 人民币兑日元大幅升值

适用：Netflix、Spotify、Adobe、日本区 App Store

优势：定价低 + 汇率贬值 = 双重省钱（省 30%–40%）

支付：Visa/Master 全币卡（无货币转换费）

### 4. 黄金：汇率对冲神器

**黄金为什么是"汇率对冲神器"？**

核心一句话：**黄金是全世界公认的"无国籍货币"，不受单一国家央行控制。**

#### 对冲逻辑

1. **黄金以美元计价，但本质不认任何货币**
   - 国际金价 = **美元/盎司**
   - 当**美元变弱** → 金价通常上涨
   - 当**人民币变弱** → 人民币金价也会涨
   - 当大家不信任法币 → 黄金涨

2. **人民币金价 ≠ 国际金价**

   人民币金价 = 国际金价（美元） × 美元兑人民币汇率

   - 如果**人民币贬值**（美元变贵）→ **人民币金价也会自动上涨**
   - 如果**人民币升值** → 金价可能承压，但黄金本身还能抗通胀

3. **乱世、通胀、危机时，黄金是"终极避险"**
   - 战争
   - 金融危机
   - 高通胀
   - 央行疯狂印钱

#### 具体对冲例子

假设你有 10 万元：

**方案A：持有人民币现金**
- 人民币突然贬值 5%
- 你在国内购买力没变，但**换成外币就少了5%**

**方案B：10%买黄金（1万），9万人民币**
- 人民币贬值 5%
- 通常**人民币金价会涨 5%~8%**
- 黄金部分赚 500~800 元，**刚好抵消汇率损失**

#### 普通人怎么"玩黄金对冲"？

**首选：黄金ETF / 黄金ETF联接基金**
- 支付宝、微信、券商都能买
- 1元起
- 费率低
- 跟踪国内金价，完美对冲人民币汇率

**银行积存金**
- 适合喜欢看得见、想慢慢攒的人
- 可以按克买，随时提取实物

**不推荐**
- 黄金首饰（溢价高、变现亏手工费）
- 黄金期货（杠杆大，普通人别碰）
- 高风险黄金QDII（容易折溢价）

#### 配置比例

**黄金不是用来发财的，是用来"防崩"的。**

- 保守型：总资产 **5%**
- 稳健型：**5% ~ 10%**
- 激进型：**最多不超过15%**

#### 黄金对冲哪些汇率场景最有效？

1. **对冲人民币贬值压力**
   - 担心国内货币宽松
   - 担心经济下行
   - 担心出口压力导致汇率走弱
   → 配点黄金，**人民币金价会自动抬升**

2. **对冲美元太强、但你又不想全仓美元**
   - 美元强时：美元资产涨，但人民币资产承压，黄金通常也偏强
   - 黄金可以让你**不用持有美元，也吃到一部分美元强势红利**

3. **对冲"美元也不稳"的时刻**
   - 美国债务爆表、美联储大放水、美国信用被质疑
   - 这时候美元可能跌，但黄金会涨

### 5. 能源：汇率对冲硬通货

能源（石油、天然气等）跟黄金一样，也是天然的「汇率对冲硬通货」，而且逻辑比黄金更直接、更粗暴。

能源 = 工业世界的血液，比黄金更刚需，对冲汇率、通胀、地缘冲突都极强。

## 组合思路

人民币偏强 + 美元强 + 日元极弱 + 全球通胀不低

这种环境下：
- **黄金**：稳，做底仓
- **能源**：弹性大，做对冲 + 进攻
- **日元资产**：超跌套利
- **美元 QDII**：吃利息 + 海外增长

组合起来就是一套完美对冲汇率波动的全球资产组合。

## 重要记录

美元强，黄金应该跌。
