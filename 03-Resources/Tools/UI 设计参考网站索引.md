# UI 设计参考网站索引

## Summary

这页收录可长期复用的 UI 设计参考网站，目的是降低找灵感、查竞品和做界面对标时的搜索成本，而不是囤积短期流量站点。

## 这页解决什么问题

- 需要快速找到某类界面参考，而不是重新全网搜索
- 需要区分“灵感站”“真实产品流程站”“后台界面站”和“设计史资料站”
- 需要把高频参考入口沉淀成稳定工具索引

## 收录原则

- 优先收录能长期复用的平台，而不是短期热点集合
- 优先回答“适合什么场景”，而不是只给链接
- 与现有条目有清晰差异，不做重复堆叠

## 分类索引

### 综合设计参考

#### `Refero`

- 定位：产品页面截图搜索引擎
- 适用场景：竞品研究、组件对标、注册后真实界面参考
- 亮点：可按行业、产品名称和组件名称检索
- 链接：https://refero.design/

#### `Reeoo`

- 定位：潮流网站设计案例库
- 适用场景：配色、风格和网页视觉灵感
- 亮点：可按颜色筛选，适合先找视觉方向
- 链接：https://reeoo.com/

### SaaS 产品专项

#### `SaaS Landing Page`

- 定位：SaaS 落地页案例库
- 适用场景：功能介绍页、定价页、营销页面参考
- 亮点：聚焦 SaaS 页面结构，不需要自己再做主题过滤
- 链接：https://saaslandingpage.com/

#### `SaasUI`

- 定位：SaaS 后台界面库
- 适用场景：CRM、ERP、数据看板、权限管理等后台设计参考
- 亮点：真实后台截图多，适合 B 端布局对标
- 链接：https://www.saasui.cc/

### 移动端设计

#### `UI Notes`

- 定位：中文移动端界面库
- 适用场景：移动端产品研究、中文语境下的交互参考
- 亮点：内容中文化，查找成本低
- 链接：https://www.uinotes.com/

#### `Mobbin`

- 定位：海外优质应用库
- 适用场景：研究完整用户流程，而不只是单张截图
- 亮点：覆盖 iOS、Android、Web，可按流程浏览
- 链接：https://mobbin.com/

### 设计系统

#### `Ant Design Pro`

- 定位：B 端设计系统与后台模板参考
- 适用场景：企业后台、设计规范、表单与复杂业务界面
- 亮点：不仅可看视觉，还能反向理解组件与业务结构
- 链接：https://pro.ant.design/

### 设计历史

#### `Web Design Museum`

- 定位：网页设计历史资料库
- 适用场景：看趋势演变、研究旧网页语言、做复古风格参考
- 亮点：能把“今天的审美”放回更长的时间尺度里看
- 链接：https://webdesignmuseum.org/

## 快速选用指南

| 场景 | 推荐 |
|---|---|
| 快速找竞品界面 | `Refero` |
| 找网页风格和配色 | `Reeoo` |
| 做 SaaS 落地页 | `SaaS Landing Page` |
| 做 B 端后台 | `SaasUI` + `Ant Design Pro` |
| 看移动端流程 | `Mobbin` |
| 看中文移动端界面 | `UI Notes` |
| 看设计历史演变 | `Web Design Museum` |
