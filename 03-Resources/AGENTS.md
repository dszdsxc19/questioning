# 03-Resources AGENTS

这个目录优先存放外部知识、工具信息、资料索引与可复用参考页。

这里的 Markdown 笔记在遵守 vault 根级 `AGENTS.md` 的前提下，补充以下局部要求：

## 页面结构

- 默认按 `99-Templates/Resource Page.md` 的模板写
- 新建页面时，优先复用该模板，而不是自由发挥结构
- 如果页面是工具索引页、主题参考页或文章摘要页，也应尽量保持同一骨架，只在“核心内容”部分按需展开

推荐保留以下 section：

- `Summary`
- `这页解决什么问题`
- `选用原则`
- `核心内容`
- `Sources`
- `Related`
- `Open Questions`

## 目标

- 保持 `03-Resources` 下页面结构稳定，降低维护成本
- 让资源页优先服务检索、复用、比较与后续更新
- 避免每篇资源页都写成不同风格，导致 wiki 网络难以维护

## 例外规则

- 如果某页只是单篇文章摘录或极短参考卡片，可以缩短正文，但仍应尽量保留最小 frontmatter
- 如果某页已经是明确的 canonical page，不要为了套模板而制造重复页面
