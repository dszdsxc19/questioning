# 03-Resources AGENTS

这个目录优先存放外部知识、工具信息、资料索引与可复用参考页。

这里的 Markdown 笔记在遵守 vault 根级 `AGENTS.md` 的前提下，补充以下局部要求：

## 基本要求

- 保持统一最小 frontmatter：`tags`、`type`、`status`、`created`、`updated`
- 页面结构以清晰、低维护、便于复用为先，不要求强制套用同一模板
- 如果某页是 canonical page，要明确它解决什么问题，避免和补充页并行竞争

## 写法原则

- 优先写成可检索、可复用、可比较的资源页
- 如果内容本身就是模板、索引、词库或操作卡片，可以按最适合复用的形式组织
- 不要为了形式统一，把有直接复用价值的内容压扁成空泛摘要

## 目标

- 保持 `03-Resources` 下页面边界清晰
- 让资源页优先服务未来复用，而不是服务当前一次性整理
- 避免制造重复主题页、空洞索引页和低价值摘录页
