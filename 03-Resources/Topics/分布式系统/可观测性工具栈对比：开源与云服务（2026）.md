# 可观测性工具栈对比：开源与云服务（2026）

## Summary

这页对比 OpenTelemetry、Prometheus、Grafana、Tempo、Loki 与阿里云 ARMS、SLS 之间的职责对应关系。它比纯产品说明更有价值的地方，是把“生产者、存储、可视化、云厂商替代层”拆开。

## 核心判断

- OpenTelemetry 更像数据生产与采集标准入口
- Prometheus / Mimir 更偏 Metrics 后端
- Tempo 更偏 Traces 后端
- Loki / SLS 更偏 Logs 平台
- Grafana 更偏最终查询与可视化层
- ARMS / SLS 是云厂商整合后的托管方案，不等于单个开源组件的一一映射
